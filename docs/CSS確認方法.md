# Tailwind CSS適用確認方法

## 1. ブラウザの開発者ツールで確認

### ステップ1: ブラウザでページを開く
1. 開発サーバーを起動: `npm run dev`
2. ブラウザで `http://localhost:3000` を開く

### ステップ2: 開発者ツールを開く
- **Windows/Linux**: `F12` または `Ctrl + Shift + I`
- **Mac**: `Cmd + Option + I`

### ステップ3: Elements/要素タブで確認
1. 要素を選択ツール（左上のカーソルアイコン）をクリック
2. ページ上の要素をクリックして選択
3. 右側の「Styles」パネルで、Tailwindクラスが適用されているか確認
   - 例: `bg-indigo-600` → `background-color: rgb(99, 102, 241);`

### ステップ4: コンピューテッドスタイルで確認
- 「Computed」タブで最終的なスタイルを確認
- Tailwindのユーティリティクラスが実際のCSSに変換されているか確認

## 2. ネットワークタブでCSSファイルを確認

1. 開発者ツールの「Network」タブを開く
2. フィルターで「CSS」を選択
3. ページをリロード（`Ctrl + R` / `Cmd + R`）
4. `globals.css` や `.next/static/css/` 配下のファイルが読み込まれているか確認
5. CSSファイルをクリックして、Tailwindのクラスが含まれているか確認

## 3. コンソールでエラーを確認

1. 開発者ツールの「Console」タブを開く
2. Tailwind CSS関連のエラーがないか確認
3. 以下のようなエラーが出ていないか確認:
   - `Module not found: Can't resolve 'tailwindcss'`
   - `PostCSS config error`

## 4. 開発サーバーのコンソールで確認

ターミナル/コマンドプロンプトで以下を確認:
- ビルドエラーが出ていないか
- `Compiled successfully` と表示されているか
- Tailwind関連の警告がないか

## 5. 簡単な視覚確認テスト

以下の要素が正しく表示されていれば、CSSは適用されています：

### ログインページ (`/`)
- ✅ グラデーション背景（indigo → purple → pink）
- ✅ 白い半透明のカード
- ✅ 丸みを帯びた角（rounded-3xl）
- ✅ 影（shadow-2xl）
- ✅ 入力フィールドにフォーカス時の青い枠線

### ダッシュボード/管理者ページ
- ✅ ナビゲーションバーのグラデーション背景
- ✅ カードの白背景と影
- ✅ グラデーションボタン
- ✅ ホバー時の色変化

## 6. 簡単なテストページを作成

`app/test/page.tsx` にテストページを作成して、基本的なTailwindクラスが動作するか確認できます。

## よくある問題と解決方法

### 問題1: スタイルが適用されない
- 解決: ブラウザのキャッシュをクリア（`Ctrl + Shift + R` / `Cmd + Shift + R`）
- 解決: 開発サーバーを再起動

### 問題2: 一部のクラスだけ適用されない
- 解決: `tailwind.config.js` の `content` パスを確認
- 解決: クラス名のスペルミスを確認

### 問題3: ビルドエラーが出る
- 解決: `node_modules` を削除して `npm install` を実行
- 解決: `.next` フォルダを削除して再ビルド
